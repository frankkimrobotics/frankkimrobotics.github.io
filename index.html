Research Projects

Differentiable packing simulation for 3D irrregular packing

  • Rapid growth of e-commerce sector in logistics domain requires advanced packing algorithm that considers irregular object shapes
  • However, existing methods (heuristics, RL) focus on accelerated collision checking between objects based on discrete shape representation
  • Fundamental challenges for e-commerce packing involves dynamic stability, object orientaiton in continuous space, and underexplored combinatorial search on object packing sequence
  • We addressed this challenges to convert packing problem into contiuous optimization using neural SDF, differentiable process, and topological graph search
